# Share A Note

Share A Note is a front-end student collaboration project built with **HTML, CSS, and JavaScript**. It was packaged as a clean GitHub portfolio piece to present to recruiters and hiring managers.

## Project overview

This application prototype is designed to help students manage shared academic work through a simple workflow:

- log in to the application
- access a dashboard for collaboration tools
- open an interactive calendar
- add task notes to specific days

The project emphasizes **UI structure, usability, responsive layout, and clear project presentation** rather than full backend functionality.

## Why this repo works for hiring managers

This version is organized to showcase:

- clean file structure
- readable front-end code
- a clear user journey across multiple pages
- responsive styling
- a polished README and demo instructions
- the ability to take a classroom project and present it professionally

## Features

- **Project landing page** with overview and feature summary
- **Demo login page** with client-side validation
- **Dashboard page** for core navigation
- **Interactive calendar page** for adding task notes to dates
- **Responsive design** for desktop and smaller screens

## How to run locally

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Click **Launch Demo**.
4. Sign in with the demo credentials above.
